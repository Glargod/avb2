Introduction

The AVB² (All-Vehicle Black Box) is a compact, rugged, subscription-free telemetry device for all vehicles: automobiles, ATVs, boats, snowmobiles, gliders, and future flying cars. It records ride logs and crash data—accessible via Wi-Fi or Bluetooth—with no cellular dependency. AVB² is the data logger and crash communicator; all visualizations, alerts, and enhanced features like gamified air time and tilt logs are delivered through the companion app.

The Challenge

Most vehicles lack robust telemetry solutions:

  • No OBD-II ports in many automobiles (e.g., classics, custom builds), ATVs, snowmobiles, boats, gliders, or flying cars.
  • Current solutions depend on expensive subscriptions.
  • Live signals fail in remote areas.
  • Lack of standardized crash data access.

SVG Visual: Table showing red "X" marks for telemetry gaps across cars, ATVs, boats, and eVTOLs.

The AVB² Solution

AVB² redefines vehicle telemetry:

  • Universal Mount: Works with all vehicles, including automobiles, ATVs, boats, snowmobiles, gliders, and flying cars.
  • Rugged Build: IP68 sealed, -40°C to 85°C, ~100g crash resistance.
  • Smart Logging: 10Hz GPS/IMU + crash buffer + event triggers.
  • Private + Flexible: Wi-Fi syncing, Bluetooth crash access, optional cloud.
  • App-Driven Features: Gamified air time, tilt logs, route visualization (via mobile app).
  • Affordable: No recurring fees, robust firmware.

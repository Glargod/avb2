Introduction

The AVB² (All-Vehicle Black Box) is a compact, rugged telemetry device designed for adventure vehicles—ATVs, snowmobiles, boats, gliders, and future flying cars. Subscription-free, it records usage data and crash telemetry, accessible via Wi-Fi or Bluetooth, with local storage and optional cloud archiving. Built to survive extreme conditions, AVB² empowers parents, rental companies, insurers, and responders with reliable data.

SVG Visual: Sleek AVB² device (rounded square with LED indicators) mounted on an ATV, with a shadow effect to suggest ruggedness.

The Challenge

Recreational and emerging vehicles lack robust telemetry solutions:

  • No OBD-II ports in ATVs, snowmobiles, boats, gliders, or flying cars.
  • Existing devices rely on costly cloud subscriptions or live radio signals, impractical for remote areas.
  • Few solutions are rugged enough for crashes, submersion, or diverse environments.
  • No standardized crash data access for insurers or first responders.

SVG Visual: Graph with red "X" marks indicating telemetry gaps across ATV, boat, glider, and eVTOL markets.

AVB² Solution

The AVB² redefines vehicle telemetry:

  • Universal Fit: Adapts to ATVs, snowmobiles, boats, gliders, and flying cars with versatile mounting.
  • Rugged Design: IP68 waterproof, ~100g crash-resistant, -40°C to 85°C.
  • Data Storage: Local storage (16MB, ~100 hours rough telemetry), optional cloud archiving via Wi-Fi.
  • Crash Analysis: Records 10Hz GPS/IMU data for 10 minutes pre-crash, with Bluetooth read-only access for responders/insurers.
  • Subscription-Free: Wi-Fi (home, office, hotspot) for routine data, no ongoing costs.
